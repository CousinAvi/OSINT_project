Main idea: getting crypto wallet balances

Module description: Often a person or organization using cryptocurrency does not have one address, but a whole multitude. This set of addresses is usually called a crypto wallet. However, despite the prevalence of this practice, There are not many sfotas that allow you to fully study the balances of a crypto wallet. This module allows you to flexibly set a crypto wallet in the form of a set of addresses of accounts and contracts of ERC-20 tokens and see the whole balances of such a crypto wallet.
